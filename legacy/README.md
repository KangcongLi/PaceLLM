# Legacy Reference Code

For new usage, prefer:

- `pacellm/activation_memory_bank.py`
- `pacellm/cortical_expert_clustering.py`
- `pacellm/qwen2_integration.py`
- `scripts/cluster_qwen2_mlp.py`
